# Personal Career Navigator - Frontend

Next.js dashboard for AI career development system.

## Features

- Profile management with file uploads
- Market intelligence analysis
- Interactive skill gap visualizations
- 30-day adaptive roadmap timeline
- Weekly evaluation tracking

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open http://localhost:3000

## Pages

- /profile - Profile setup and data upload
- /market - Job market analysis for target role
- /gaps - Skill gap visualization and analysis
- /roadmap - 30-day learning roadmap
- /evaluation - Weekly evaluation and progress tracking

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts for data visualization
- Framer Motion for animations
- Dark theme professional design

## Build
```bash
npm run build
npm start
```
